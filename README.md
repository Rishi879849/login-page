# 🌐 Dynamic Login & Registration Page

A sleek, modern, and fully responsive User Authentication interface featuring a beautiful cosmic design, interactive form validation, and smooth layout switching. This application serves as a complete front-end solution for user login and registration workflows.

---

## ✨ Features

*   **🔒 Secure Input Layouts:** Separate, dedicated interfaces for seamless user log-in and account registration.
*   **🌌 Cosmic UI Styling:** Powered by custom stellar background graphics (`nebula-bg.png`) and branded iconography (`dtv-logo.png`).
*   **📱 Fully Responsive:** Fully optimized across desktop, tablet, and mobile displays.
*   **⚡ Active Form Validation:** Real-time client-side interactivity ensuring users input correctly formatted criteria before submission.
*   **🎨 Clean Architecture:** Strict separation of concerns across structural markup, visual stylesheets, and procedural logic.
